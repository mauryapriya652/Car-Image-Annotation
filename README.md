# 🚗 Car Image Annotation Tool

A Python-based image annotation tool used to create bounding box annotations for car images in YOLO format.

## Features

- Draw bounding boxes using the mouse
- Save annotations in YOLO format
- Supports multiple images
- Easy to use
- Built with Python and OpenCV

## Project Structure

```
Car-Image-Annotation/
│
├── car_dataset/
│   ├── images/
│   └── labels/
│
├── annotate.ipynb
├── README.md
└── .gitignore
```

## Technologies Used

- Python
- OpenCV
- Jupyter Notebook

## Output

The tool generates `.txt` annotation files in YOLO format for every annotated image.
